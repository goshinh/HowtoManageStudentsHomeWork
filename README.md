# HowtoManageStudentsHomeWork

## 适用于什么情境？
作为一名信息技术老师，您可能会身处这样的困境中：

- 班级多

担任很多个班级的信息技术课程
- 学生多

每个班级学生人数众多
- 任务重

忙于很多事务，时间很有限

**如何快速统计每个班每节课哪些同学提交了作业，哪些同学没有？**
## 优势
针对以上情况，这个项目提供了一种解决方案。

不用安装任何软件，不需任何特殊的编程语言——所需的就是MS-DOS

每次统计学生交作业情况时，只需运行一个命令，就这么简单。
